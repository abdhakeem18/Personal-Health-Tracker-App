# HealthMate - Personal Health Tracker

A Flutter app for tracking daily health activities including steps, calories, and water intake.

## Features
- Daily health tracking (steps, calories, water)
- Progress visualization with charts
- Goal setting and tracking
- Dark mode support
- Custom themes

## Getting Started

Run the app:
```bash
flutter run
```

## Database
Uses SQLite for local data storage.

## Dependencies
- provider (state management)
- sqflite (database)
- fl_chart (data visualization)
- intl (date formatting)
